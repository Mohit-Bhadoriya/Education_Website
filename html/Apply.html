BEFORE YOU APPLY:


General Instructions:

• Please read the FAQs section before proceeding with the application form.

• Recommended browser to fill the form:


Microsoft Edge (version 17) or
Mozilla Firefox (version 50 and above) or
Google Chrome (version 50 and above) to fill in the Application Form.

• Notes, Instructions and Important Links will guide you to fill the online application accurately. Please read them carefully before making your choice.

• Use mouse to move between fields for entry of data instead of using the tab key.


Instructions for Registration:

• You require a mobile number for registration. Once registered the same number cannot be used for a different registration and cannot be changed.

• Fill the registration form for obtaining User ID and password.

• Fields marked with * are mandatory.

• Fill in all the required details and click on Submit to proceed.

• Your User ID and password will be sent to the registered mobile number through SMS and to your registered email address.

• You are encouraged to change your password the first time you log in to the system using the credentials sent to you.


Instructions for Application:

Documents:

Kindly be ready with the soft/scanned copies of the following before filling the application form:


1) Photo (max size: 200 KB, format supported: Only JPEG and JPG)

2) Signature (max size: 200 KB, format supported: Only JPEG and JPG)

3) Mark sheet of 12th class or equivalent (candidates appearing for exam in 2021 may upload later) (maximum size: 1 MB, format supported: Only PDF, JPEG and JPG)

4) Category certificate (if applicable) (maximum size: 1 MB, format supported: Only JPEG, JPG and PDF).


You should also upload the following documents in the application form, if applicable. In case you can’t upload these documents at this point of time, please keep them ready. You may be asked to provide them at a later point.

1. Photo ID (not mandatory now, will be needed later) (maximum size: 200 KB, format supported: Only JPEG, JPG and PDF).

2. Kashmiri Migrant certificate (not mandatory now, will be needed later if applicable) (maximum size: 1 MB, format supported: Only PDF, JPEG and JPG).

3. Medical certificate for PwD (not mandatory now, will be needed later if applicable) (maximum size: 1 MB, format supported: Only PDF, JPEG and JPG).

4. OCI/PIO certificate (not mandatory now, will be needed later if applicable) (maximum size: 1 MB, format supported: Only PDF, JPEG and JPG).


NEW> In case, you do not have the category certificate readily available - but expect it soon, you can upload the self undertaking [click here] instead. However, you will have to upload the requisite certificate during the admission, if selected. This is a special exemption provided due to the COVID-19 pandemic.


NEW> For photo-ID, if none of the other listed documents are available, the Class X marksheet/certificate must be available to you. You may use it as a photo-ID while filling up the application form even if it doesn’t have your photo. At the examination centre and during admission, if selected, you can use Aadhaar as the proof of identification.


Application fee:

Please note that the application will not be complete unless the application fee is paid online – as the form will direct you to. There is a non-refundable application fee of Rs.2,000/- for candidates applying under General, EWS, and OBC-NCL categories, and Rs.1,000/- for Persons with Disabilities and candidates under SC/ST category. The application fee for foreign nationals is USD 100.
